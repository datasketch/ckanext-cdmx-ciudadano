¿Cómo cargar mis archivos?

Puedes subir archivos directamente desde tu ordenador o añadir un enlace al archivo almacenado en la nube como Google Drive.

Es altamente recomendable añadir un diccionario de datos, notas metodológicas, archivo de interpretación de claves o cualquier otro recurso que ayude a las personas a entender la información que compartes.

Si no sabes qué debe contener un diccionario de datos puedes descargar nuestra plantilla en el siguiente enlace:

Plantilla de diccionario de datos

¿Quieres que tus datos se puedan cruzar con otros?

Para que tus datos puedan ser cruzados con otros datos es necesario que tu base cuente con al menos una de las variables de cruce y que esté en el formato adecuado. Las variables por las que es posible cruzar bases de datos son: fecha, alcaldía, colonia, localidad o AGEB.

¿Cómo funcionan los visualizadores de datos?

Existen cuatro tipos de visualizadores de datos que se adaptan a diferentes formatos o a la información que contengan tus archivos. Cada visualizador puede configurarse para que las visualizaciones mostradas sean las más idóneas.
